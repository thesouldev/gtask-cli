# gtask docs site

The documentation site for gtask, built with Astro Starlight and the
lucode-starlight theme. Published to GitHub Pages at
https://thesouldev.github.io/gtask-cli.

## Develop

Requires Node 22.12 or newer.

```bash
npm install
npm run dev        # http://localhost:4321/gtask-cli/
```

From the repository root you can also use `make site` and `make site-build`.

## Content

Pages live in `src/content/docs/`:

- `index.mdx` the landing page
- `getting-started/setup.md` install and OAuth setup
- `reference/usage.md` the CLI reference, generated from the code

The CLI reference is generated by `make docs` from the repository root, which
runs the Typer docs command and writes `reference/usage.md`. Do not edit that
file by hand.

## Commands

| Command | Action |
| --- | --- |
| `npm run dev` | start the dev server |
| `npm run build` | build the static site to `dist/` |
| `npm run preview` | preview the build locally |

## Deploy

Pushing changes under `site/` to `master` triggers
`.github/workflows/deploy.yml`, which builds the site and publishes it to
GitHub Pages.
